Gravity Flux

2D игра - платформер, с изменением гравитации. В игре есть несколько уровней, к каждому уровню различные персонажи.

Этапы:
1 Заставка
    1.1 Текст - Наименование игры
    1.2 Отрисовка различных геометрических фигур

2 Окно меню
    2.1 Кнопка "Играть", которая открывает окно меню выбора уровня
        2.1.1 Кнопка выбора уровня 'easy'
            2.1.1.1 Кнопка выбора первого персонажа "Кассиан"
                2.1.1.1.1 Формирование уровня, путём подгрузки дополнительной информации связанной с выбором пользователя
            2.1.1.2 Кнопка выбора второго персонажа "Золтан"
                2.1.1.2.1 Формирование уровня, путём подгрузки дополнительной информации связанной с выбором пользователя
            2.1.1.3 Кнопка выбора третьего персонажа "Келтор"
                2.1.1.3.1 Формирование уровня, путём подгрузки дополнительной информации связанной с выбором пользователя
        2.1.2 Кнопка выбора уровня 'normal'
            2.1.2.1 Кнопка выбора первого персонажа "Рен"
                2.1.2.1.1 Формирование уровня, путём подгрузки дополнительной информации связанной с выбором пользователя
            2.1.2.2 Кнопка выбора второго персонажа "Эйден"
                2.1.2.2.1 Формирование уровня, путём подгрузки дополнительной информации связанной с выбором пользователя
            2.1.2.3 Кнопка выбора третьего персонажа "Киран"
                2.1.2.3.1 Формирование уровня, путём подгрузки дополнительной информации связанной с выбором пользователя
        2.1.3 Кнопка выбора уровня 'hard'
            2.1.3.1 Кнопка выбора первого персонажа "Финн"
                2.1.3.1.1 Формирование уровня, путём подгрузки дополнительной информации связанной с выбором пользователя
            2.1.3.2 Кнопка выбора второго персонажа "Корвус"
                2.1.3.2.1 Формирование уровня, путём подгрузки дополнительной информации связанной с выбором пользователя
            2.1.3.3 Кнопка выбора третьего персонажа "Блейв"
                2.1.3.3.1 Формирование уровня, путём подгрузки дополнительной информации связанной с выбором пользователя
    2.2 Кнопка "Настройки", которая открывает окно нвстроек
        2.2.1 Отображение вспомогательногго текста, для лучшей оринтации пользователя
        2.2.2 Кнопка "Включить/Выключить", которая останавливает или возобновляет проигрывание музыки
        2.2.3 Слайдер, который регулирует громкоть музыки, если она не приостановлена, иначе, он не отображается
        2.3.4 Кнопка "Включить/Выключить", которая останавливает или возобновляет проигрывание звуковых эфектов
        2.2.5 Слайдер, который регулирует громкоть звуковых эфектов, если они включены, иначе, он не отображается
        2.2.6 Кнопка "Назад", которая возвращает пользователя обратно в главное меню
    2.3 Кнопка "Выход", котоорая завершает работу системы (игры)

3 Игровой процесс
    3.1 Игрок, с собственной анимацией
    3.2 Враги, с различныыми типами оружий
    3.3 Управление
        3.3.1 Кнопка "D" - передвижение игрока вправо
        3.3.2 Кнопка "A" - передвижение игрока влево
        3.3.5 Кнопка "Q" - атака игрока
        3.3.6 Кнопка "SPACE" - прыжок персонажа
        3.4.6 Кнопка "E" - изменение гравитации игрока
    3.4 Кнопка "Настройки", которая открывает окно нвстроек
        3.4.1 Отображение вспомогательногго текста, для лучшей оринтации пользователя
        3.4.2 Текст - подсказка управления персонажем
        3.4.3 Кнопка "Вернуться", которая возвращает пользователя в главное меню
        3.4.2 Кнопка "Включить/Выключить", которая останавливает или возобновляет проигрывание музыки
        3.4.3 Слайдер, который регулирует громкоть музыки, если она не приостановлена, иначе, он не отображается
        3.4.4 Кнопка "Включить/Выключить", которая останавливает или возобновляет проигрывание звуковых эфектов
        3.4.5 Слайдер, который регулирует громкоть звуковых эфектов, если они включены, иначе, он не отображается
        3.4.6 Кнопка "Назад", которая возвращает пользователя обратно в игровой процесс (уровень)

4 Экран окончания (завершения) уровня
    4.1 Подвод результатов игры
        4.1.1 Подсчёт количества заработанных очков
    4.2 Кнопка выхода (возврата) в главное меню
